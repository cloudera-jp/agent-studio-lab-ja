# ワークショップ範囲

**時間計画**：セッション 1（90分）+ セッション 2（2時間）+ セッション 3（2時間）+ まとめ・次のステップ（30分）

## **セッション 1：AI エージェント・エージェンティックワークフロー基礎（プレゼンテーション）**

目的：AI エージェント、その構成要素、および適用例の基本的な理解を構築する。  
時間：90分

### **1.1 ようこそ・ワークショップの目的（15分）**

* ワークショップと議題の簡単な紹介
* 地域リーダーシップビデオ / 対面コメント

### **1.2 エージェントワークフロー設計への導入（30分）**

* エージェントワークフロー定義と従来のワークフローとの違い
* エージェント設計ワークフローにおける設計上の考慮事項
* ワークフロー設計演習 - グループによる協調作業

### **1.3 産業応用 - AI エージェント実践例（30分）**

* 産業応用における AI エージェントの例（例：カスタマサポート、ワークフロー自動化、RAG 対応アシスタント）
* 従来のプログラミングと AI 駆動型エージェントの比較

### **1.4 AI エージェント・アーキテクチャ（45分）**

* AI アーキテクチャ・エージェント
* 概念的構成：バックストーリー、目標、説明／期待される出力

**休憩（30分）**

## **セッション 2：ハンズオンラボ パートA**

目的：AI エージェントを手動で実装し、その内部動作を理解する。  
時間：2時間

### **2.1 ラボセットアップと準備（30分）**

* Share the use case: AirAware  
* Use CAI to clone the repo  
* Set up Python environment, install necessary libraries (Crew AI, OpenAI API, etc.), and provide datasets.

### **2.2 Building the Agent by Hand (2 Hours)**

* **Step 1: Design the agent architecture.**  
  * Define input-output processes.  
  * Define Agent Roles, Tasks, Tools and Process  
* **Step 2: Implement the agent in Python.**  
  * Build Custom Tools  
  * Test the agent’s behavior with various inputs.

## **Session 3: Hands-On Lab Part B**

Objective: Use Cloudera Agent Studio for building the same AI Agent workflow  
Duration: 2 Hours

### **3.1 Introduction to Cloudera Agent Studio (15 minutes)**

* Overview of Cloudera Agent Studio and its features.  
* Benefits of a no-code/low-code approach to AI Agents.

### **3.2 Building the Agent with Cloudera Agent Studio (1 Hours)**

* Rebuild the Use case  
* Define agent workflows in the visual interface.  
* Configure integrations with external tools and APIs.  
* Deploy and test the agent within Cloudera.  
* Highlight efficiency improvements compared to manual implementation.

### **3.3 Comparison and Reflections (30 minutes)**

* Compare manual implementation with Cloudera Agent Studio.  
* Discuss scalability, ease of maintenance, and extensibility.

## **Session 4: Wrap-Up and Next Steps**

**Duration**: 30 Minutes

### **4.1 Advanced Concepts in Building Agentic Workflows**

* Pros & Cons to consider e.g. Token Size increases Dramatically  
* Patterns of Agentic Workflows ( Reflection, Self correction etc.)  
* Quality of Output based on the LLM Used Debugging & Error handling  
* Scaling Workflows : Considerations

### **4.2 Key Takeaways (15 minutes)**

* Mapping to Industry Use Cases ( applying APIs, synthesizing multiple agent data etc.)  
* Summarize key lessons from the workshop.  
* Capstone Project Details
